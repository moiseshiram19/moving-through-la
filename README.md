# Moving Through LA

An interactive analysis of public transportation, bike share systems, 
and demographics in Los Angeles. Answering the questions if there is a demand why can we get access to more infraestructure?

## Data Sources
- LA Metro API
- Metro Bike Share
- US Census / ACS

## Goalssss
- Understand mobility access across neighborhoods
- Identify gaps between transit supply and demographic demand
- Build an interactive map dashboard
- 

## Tech Stack
- Python (geopandas, pandas)
- React + Deck.gl
- US Census API
